# Expense Tracker 
This is simple console base java project to track expenses using Collection Framework, Exception handling , OOPs
<h4>Features</h4>
<ul>
  <li>
    Login
  </li>
   <li>
    Add Expenses
  </li>
   <li>
   Delete Expense
  </li>
   <li>
    View Expenses
  </li>
   <li>
    Search Expenses by category
  </li>
   <li>
    View Summary
  </li>
    <li>
   View Expense Report (Weekly, Monthly, Yearly)
  </li>
    <li>
   View History
  </li>
     <li>
Logout
  </li>
</ul>
